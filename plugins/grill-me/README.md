# Grill Me Plugin

Interview the user relentlessly about a plan or design until reaching shared understanding, resolving each branch of the decision tree.

**Source:** [mattpocock/skills](https://github.com/mattpocock/skills/tree/main/grill-me)

## Usage

Use `/grill-me` or mention "grill me" when you want to stress-test a plan or design.

## Installation

```bash
/plugin marketplace add bryan-cox/personal-claude-skills
/plugin install grill-me@personal-claude-skills
```
