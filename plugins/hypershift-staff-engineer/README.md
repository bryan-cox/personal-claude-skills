# HyperShift Staff Engineer Plugin

Expert-level code review, architecture guidance, and best practices for HyperShift development.

## What It Does

This plugin provides an agent that embodies the collective expertise of senior HyperShift engineers. It applies their review standards when:

- Reviewing pull requests with comprehensive, prioritized feedback
- Evaluating design decisions and operator placement
- Assessing API changes for CRD evolution, CEL validation, and backward compatibility
- Reviewing controller implementations for reconciliation patterns and idempotency
- Advising on testing strategies

## Installation

### From Claude Code Plugin Marketplace

```bash
/plugin marketplace add bryan-cox/personal-claude-skills
/plugin install hypershift-staff-engineer@personal-claude-skills
```
